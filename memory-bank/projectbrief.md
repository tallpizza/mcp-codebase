# MCP Codebase 프로젝트

## 프로젝트 개요

Model Context Protocol (MCP)을 구현한 코드베이스 관리 서버입니다. 이 서버는 코드베이스를 분석하고, 코드 청크를 관리하며, 프로젝트 단위로 코드를 구조화하여 저장합니다.

## 핵심 요구사항

1. 프로젝트 단위의 코드베이스 관리
2. 코드 청크 단위의 분석 및 저장
3. 벡터 기반 코드 검색 지원
4. MCP 프로토콜 준수
5. 실시간 코드 분석 및 업데이트

## 목표

- 효율적인 코드베이스 관리
- 정확한 코드 청크 추출 및 저장
- 빠른 코드 검색 및 조회
- 안정적인 데이터베이스 연동
- 확장 가능한 아키텍처 구현

## 주요 기능

1. 프로젝트 관리

   - 프로젝트 생성 및 조회
   - 프로젝트 메타데이터 관리

2. 코드 청크 관리

   - 코드 청크 추출 및 저장
   - 의존성 분석 및 관리
   - 벡터 임베딩 저장

3. 검색 기능

   - 코사인 유사도 기반 검색
   - 메타데이터 기반 검색
   - 프로젝트 단위 검색

4. MCP 도구
   - 프로젝트 도구
   - 파일 도구
   - 코드 도구

## 기술 스택

- **백엔드 프레임워크**: Hono
- **런타임**: Bun
- **언어**: TypeScript
- **벡터 임베딩**: OpenAI API
- **코드 분석**: LSP(Language Server Protocol)

## 개발 우선순위

1. 기본 서버 설정 및 라우팅 구성
2. 파일 시스템 접근 기능 구현
3. LSP 통합 및 코드 청킹 기능 구현
4. 임베딩 생성 및 저장 기능 구현
5. 벡터 검색 엔드포인트 구현
